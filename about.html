Welcome to my Markdown Blog



So Why Markdown?

Well, I didn't really have the budget for another web server or domain, and I found existing flat file blog platforms like Jekyll to be overly complicated.

The idea was to write posts as Markdown files in my IDE, where I can easily preview them, and push the content to GitHub for free hosting. A custom domain can always be added later.



Who am I?
